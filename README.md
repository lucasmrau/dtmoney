## Description

DTmoney is a small personal finance website. May include expenses and gains. It calculates the total automatically.

## Tecnologies, Libraries and Hooks

- ReactJS
- TypeScript
- useState
- useEfect
- Context API
- Styled Components
- Mirage JS
- Axios
- React Modal
- Polished
